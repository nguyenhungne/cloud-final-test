MyMiniCloud System

Hệ thống Cloud Platform mô phỏng

Chào mừng đến với MyMiniCloud

MyMiniCloud là hệ thống mô phỏng cloud platform cơ bản, bao gồm 9 loại máy chủ chạy trong các container Docker riêng biệt.

Các thành phần hệ thống:

  • Web Frontend Server (Nginx)
  • Application Backend Server (Flask API)
  • Relational Database Server (MariaDB)
  • Authentication Identity Server (Keycloak)
  • Object Storage Server (MinIO)
  • Internal DNS Server (Bind9)
  • Monitoring Server (Prometheus)
  • Visualization Server (Grafana)
  • API Gateway Proxy Server (Nginx)
