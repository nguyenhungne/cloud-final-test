Docker là gì?

Docker là một nền tảng mã nguồn mở cho phép đóng gói, phân phối và chạy ứng dụng trong các container. Container là một đơn vị phần mềm tiêu chuẩn hóa, đóng gói code và tất cả các dependencies để ứng dụng có thể chạy nhanh chóng và đáng tin cậy từ môi trường này sang môi trường khác.

Tại sao sử dụng Container?

Container giải quyết vấn đề kinh điển "works on my machine" bằng cách:

  • Tính nhất quán: Đảm bảo ứng dụng chạy giống nhau trên mọi môi trường
  • Tính di động: Dễ dàng di chuyển giữa các môi trường development, testing, production
  • Hiệu quả: Sử dụng tài nguyên hiệu quả hơn so với máy ảo truyền thống
  • Tốc độ: Khởi động nhanh chóng, chỉ trong vài giây

Docker trong MyMiniCloud

Hệ thống MyMiniCloud sử dụng Docker để chạy 9 services khác nhau trong các container riêng biệt:

  • Web Frontend Server (Nginx)
  • Application Backend Server (Flask)
  • Database Server (MariaDB)
  • Authentication Server (Keycloak)
  • Object Storage Server (MinIO)
  • DNS Server (Bind9)
  • Monitoring Server (Prometheus)
  • Visualization Server (Grafana)
  • API Gateway/Proxy Server (Nginx)

Lệnh Docker cơ bản

Một số lệnh Docker thường dùng:

  • docker ps - Xem các container đang chạy
  • docker images - Xem danh sách images
  • docker compose up -d - Khởi động hệ thống
  • docker compose down - Dừng hệ thống
  • docker logs <container> - Xem logs của container
