Monitoring là gì?

Monitoring là quá trình thu thập, phân tích và sử dụng thông tin về hiệu suất và trạng thái của hệ thống để đảm bảo nó hoạt động đúng cách. Trong môi trường cloud và microservices, monitoring là yếu tố quan trọng để duy trì độ tin cậy và hiệu suất.

Observability vs Monitoring

Monitoring trả lời câu hỏi "Có vấn đề gì đang xảy ra không?"

Observability trả lời câu hỏi "Tại sao vấn đề đó xảy ra?"

Observability bao gồm 3 trụ cột chính:

  • Metrics: Dữ liệu số đo lường hiệu suất (CPU, memory, request rate)
  • Logs: Bản ghi chi tiết các sự kiện trong hệ thống
  • Traces: Theo dõi request qua các services khác nhau

Prometheus - Time Series Database

Prometheus là hệ thống monitoring mã nguồn mở được thiết kế cho môi trường cloud-native:

  • Pull-based metrics collection
  • Time series database với PromQL query language
  • Service discovery tự động
  • Alerting rules và notification

Grafana - Visualization Platform

Grafana là nền tảng visualization và analytics:

  • Tạo dashboard trực quan với nhiều loại charts
  • Hỗ trợ nhiều data sources (Prometheus, InfluxDB, Elasticsearch...)
  • Alerting và notification
  • Template variables cho dashboard động

Monitoring trong MyMiniCloud

Hệ thống MyMiniCloud sử dụng stack Prometheus + Grafana:

  • Node Exporter: Export system metrics (CPU, memory, disk, network)
  • Prometheus: Scrape và lưu trữ metrics từ Node Exporter
  • Grafana: Visualize metrics với dashboard

Metrics quan trọng cần theo dõi

  • System Metrics: CPU usage, memory usage, disk I/O, network traffic
  • Application Metrics: Request rate, error rate, response time
  • Business Metrics: Active users, transactions, revenue
  • Container Metrics: Container count, restart count, resource limits

Best Practices

  • Định nghĩa SLIs (Service Level Indicators) và SLOs (Service Level Objectives)
  • Thiết lập alerting rules hợp lý, tránh alert fatigue
  • Sử dụng dashboard để visualize metrics quan trọng
  • Implement distributed tracing cho microservices
  • Centralized logging với ELK stack hoặc Loki
  • Regular review và optimize monitoring strategy

Truy cập Monitoring trong MyMiniCloud

  • Prometheus UI: http://localhost:9090
  • Grafana Dashboard: http://localhost:3000 (admin/admin)
  • Node Exporter Metrics: http://localhost:9100/metrics
